## 📒 Descrição

Este é dos desafios do modulo 2 do curso de HTML5 e CSS3 do Curso em Video. A proposta foi replicar uma página com um cabeçalho com tons degradê, com um menu simples e um artigo com títulos, imagens, vídeo e um conteúdo relacionado. O conteúdo também vai apresentar uma lista com marcadores personalizados e links externos identificados com um emoji. Por fim, temos um rodapé com links. O projeto pode ser visualizado em celulares e dispositivos com a tela pequena
